# AI Strategy Tools

Two practical decision tools for teams adopting AI:

- **RAG vs Fine-tuning** — a decision framework for choosing the right approach for a given use case.
- **EU AI Act Navigator** — work out how the EU AI Act applies to your system and what obligations follow.

## Tech stack

Static HTML / JavaScript, deployed on Vercel.

## Run locally

Open `index.html`, or serve the folder with any static server (e.g. `npx serve`).
